## 실습 시, 유용한 Tips
- Copy & Paste : 기본적으로 우분투에서 마우스 더블클릭 시 자동으로 복사되며, 붙여넣기는 마우스 오른쪽 버튼 클릭
- tmux 사용 시, 주요 치트키
  - 창 수직으로 나누기 : Ctrl + b + %
  - 창 수평으로 나누기 : Ctrl + b + "
  - 창 간에 이동하기 : Ctrl + b + 화살표
  - 나뉘어진 창 닫기 : Ctrl + d
  - 최종 tmux 종료하기 : Ctrl + d
